# Gametracker plugin for Joomla
Allows you to add statistic game banners to your Joomla comtent

**Supports:**

*Gametracker.com*

Use tags
 
**{gt}ip:port{/gt}**

**{gt=big}ip:port{/gt}**

**{gt=banner}ip:port{/gt}**

**{gt=block}ip:port{/gt}**

**{gt=html}ip:port{/gt}**
