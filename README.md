# Gamestat plugin for Joomla
Allows you to add statistic game banners to your Joomla content

**Supports:**

*Gametracker.com*

Use tags
 
**{gt}ip:port{/gt}**

**{gt=big}ip:port{/gt}**

**{gt=banner}ip:port{/gt}**

**{gt=block}ip:port{/gt}**

**{gt=html}ip:port{/gt}**

*Gamestate.com*

Use tags
 
**{gt}ip:port{/gt}**

**{gt=big}ip:port{/gt}**

**{gt=banner}ip:port{/gt}**

**{gt=block}ip:port{/gt}**

**{gt=userbar}ip:port{/gt}**

To include banner to your article
